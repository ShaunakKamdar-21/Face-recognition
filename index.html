Read out the prediction

Description : Use speech synthesis to read the predicition result

  1. Define a variable to store the speech API
  2. Store the label of prediction result which we want the system to speak in a variable
  3. Use the SpeechSynthesisUtterance() function to convert text to speech, pass the variable which holds the prediction result to the SpeechSynthesisUtterance() as a parameter and store it in a variable
  4. Then use speak() function and pass the variable which holds the text to speech converted data to the speak() function
  5. Write all the above code inside gotResult() function in sketch.js
  6. For reference - To recall speech API refer to the JS code of class 100(Take My Selfie app)
